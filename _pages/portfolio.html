Web Scraping and Sentiment Analysis of Steam Game Reviews

This project extracts and analyzes Steam game reviews using Python, Pandas, Selenium, NLTK, and Plotly. Selenium scrapes reviews, Pandas manipulates data, NLTK handles natural language processing including sentiment analysis, and Plotly visualizes insights like review distribution and polarity scores.
